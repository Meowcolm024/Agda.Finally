# Agda, Finally...

A collection of formalizations of **simply typed lambda calculus** with _well scoped deburijn index_ in Agda, ... with a slightly fancier frontend.

## Building

Requirements:

- `agda-2.8.0`
- `agda-stdlib-2.3`
- `node`

```sh
$ npm install    # install dependencies
$ npm run build  # build site
$ npm run serve  # start preview server
```
